# 중구 · 대구광역시

> 중구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-daegu-jung.github.io/**

대구광역시 중구는 조선시대 경상감영이 자리하며 영남의 행정·상업 중심으로 이름을 떨친 대구의 원도심이다. 동성로와 반월당을 축으로 백화점·패션 상가·먹자골목이 촘촘히 이어져 지금도 대구 최대의 도심 상권으로 손꼽힌다. 서문시장과 약령시(약전골목), 방천시장을 비롯한 전통 시장이 근대 골목의 붉은 벽돌 건물, 선교사 주택과 어우러져 도시 여행의 색을 짙게 만든다. 100년 넘은 계산성당과 청라언덕, 김광석다시그리기길처럼 근대와 대중문화가 겹쳐진 명소가 걸어서 닿는 거리에 밀집해 있다. 면적은 대구 8개 자치구 가운데 가장 작지만, 도시철도 두 노선이 교차하는 반월당역을 품어 하루 유동 인구가 가장 두터운 곳이다.

## 주요 통계

- 면적: 7.06 km²
- 신설: 1963년
